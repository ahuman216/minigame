To play a game, pick any game you would like and click the start button

Once you receive confirmation that your game has started, you can start playing!

HAVE FUN!


Password Hack

This is a game to test out your hacking skills! There is a secret passcode and YOU have to guess it in the least amount of guesses! If a number shows up with a green check mark, it is in the correct place, if it has a yellow dot, it is in the password, but in the wrong place, and if there is a red cross, it is not a part of your password! Happy guessing!

Rock Paper Scissors

This is a fun version of the classic rock paper scissor game, but instead of playing against another person, you are playing against the computer. Enter your pick, and first to 3 wins!

Hangman

This is a classic game, you have until the man is hung to keep guessing letters and try to guess the word. Just enter a letter to begin guessing to save the man!
